# Novi mediji i web tehnologije

Materijali za kolegij **Novi mediji i web tehnologije** (KOMP4-3), Komunikologija, HKS, ljetni semestar 2025/2026.

## O repozitoriju

Ovaj repozitorij sadrži bilješke s predavanja, seminarske materijale, popis literature i upute za projekt. Stranica je izrađena pomoću [Quarto](https://quarto.org) i automatski se objavljuje na GitHub Pages.

**Aktivna stranica:** [https://yourusername.github.io/web-tech-course/](https://yourusername.github.io/web-tech-course/)

## Lokalni razvoj

```bash
# Pregledaj stranicu lokalno s automatskim osvježavanjem
quarto preview

# Izgradi cijelu stranicu
quarto render
```

## Struktura

```
lectures/       Bilješke s predavanja, jedna datoteka po tjednu
seminars/       Seminarski materijali za radionice
resources/      Popis literature, alati, skupovi podataka
projects/       Upute za praktični projekt i rubrika
assets/         Slike, prezentacije i drugi mediji
```
